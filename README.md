# Save Me

A to-do app designed to help reduce stress and anxiety by tracking task energy and anxiety levels.

## Features

- **Add tasks** with custom energy and anxiety levels
- **Track progress** with a real-time stress score
- **Delete completed tasks** to clear mental clutter
- **Persist data** via local storage
- **Clean, intuitive UI** built with modern React

## Tech Stack

### Frontend
- **Next.js 16.1.6** - React framework with App Router
- **React 19.2.4** - UI library
- **TypeScript 5** - Type-safe development
- **Tailwind CSS 4.1.18** - Styling (via @tailwindcss/postcss)

### Development
- **ESLint 9** - Code quality
- **Turbopack** - Fast build system
